# Project-IAC

Este é um projeto de estudo de DevOps, utilizando um framework Node.js como base para a aplicação.

## Descrição

Este é um repositório inicial de um projeto em TypeScript.

## Instalação

```bash
$ npm install
```

## Executando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
